# Coffee Shop Website ☕

A modern coffee shop showcase website built with React and Node.js.

## Features

* Responsive design
* Home page
* Menu page with category filtering
* Gallery page with category filtering
* About page
* Reservation form
* Contact form
* Email notifications using Nodemailer
* Mobile-friendly navigation

## Tech Stack

### Frontend

* React
* React Router
* Bootstrap Icons
* Axios
* React Toastify

### Backend

* Node.js
* Express
* Nodemailer
